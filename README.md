## 💰 Personal Financial Tracker 

Track your income, expenses, and savings effortlessly with this intuitive web app! Keep your finances organized and make smarter financial decisions.                          

## 🌟 Features

Add, Edit, and Delete income and expenses

Visual Charts to monitor spending trends

Monthly Summary showing total income, expenses, and savings

Responsive Design – works on both desktop and mobile

Easy to Use with a clean, modern interface



## 🚀 Live Demo

Check it out here: Your GitHub Pages Link https://mr-danix.github.io/P.F.T/

## 🛠️ Technologies Used

Frontend: HTML, CSS, JavaScript

Storage: LocalStorage (for data persistence)

Charts: Chart.js for data visualization

## 📈 How to Use

Open the web app in your browser

Add your income and expenses using the forms

View your spending trends in interactive charts

Monitor your monthly summary and adjust your budget accordingly
